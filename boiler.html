philosophy

I always find myself using different scss setups for various projects. Burger is a combo of module sets I've used; built on a base.

Burger is intended to be a quick start scss template; useful for rapid prototyping and building an initial base for your project.

Burger isn't seasoned yet (v0.9), feel free to contribute any new condiments or toppings.
