about burger.scss

Burger is a lightweight starter framework built using Flexbox and SCSS.

Used to create starter SCSS for:

  • HTML Boilerplate
  • HTML Boilerplate
  • Ionic
  • Reddit
  • Drupal 8 (In progress)
  • Wordpress (In progress)

Setup

Clone from git. Run npm install to install local packages.

Run gulp watch to compile for local use. Enables source maps and browsersync autoreload.

Run gulp to compile for production. Enables minification.

Run browser-sync start --server --files "css/*.css" run local browsersync.

philosophy

I always find myself using different scss setups for various projects. Burger is a combo of module sets I've used; built on a base.

Burger is intended to be a quick start scss template; useful for rapid prototyping and building an initial base for your project.

Burger isn't seasoned yet (v0.9), feel free to contribute any new condiments or toppings.
